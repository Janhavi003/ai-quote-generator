# Daily AI Quote ✨

A minimal web app that presents a thoughtful AI-generated quote each day.  
Designed as a calm daily space for reflection.

Built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

---

## Features

- Daily AI-generated quote
- Shuffle to explore more quotes
- Return to today’s quote
- Download quote as an image
- Clean, light-only UI

---

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- OpenAI API
- html-to-image

---
